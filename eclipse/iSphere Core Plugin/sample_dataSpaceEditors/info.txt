Sample Data Space Editors
-------------------------

The content of this folder is automatically copied to the following
location, presumed the user has confirmed that:

  [workspace]\.metadata\.plugins\biz.isphere.core\dataSpaceEditors

The example editors have been assigned to the following example
objects as shown below:

  Editor              Object
  --------------      ----------------------
  HYBRID1             TSTDTAARA1 (*DTAARA)
                      TSTUSRSPC1 (*DTAARA)
  TSTDTAARA1          TSTDTAARA1 (*DTAARA)
  TSTUSRSPC1          TSTUSRSPC1 (*DTAARA)

The editor "HYBRID1" has both objects assigned and has a less
fields than the other editors.
